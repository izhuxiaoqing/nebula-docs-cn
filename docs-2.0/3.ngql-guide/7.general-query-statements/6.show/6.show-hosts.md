# SHOW HOSTS

`SHOW HOSTS`语句可以显示Graph、Storage、Meta服务主机信息、版本信息。

## 语法

```ngql
SHOW HOSTS [GRAPH | STORAGE | META];
```

不添加服务名，直接使用`SHOW HOSTS`，会显示Storage服务主机信息，以及leader总数、leader分布和分片分布。
